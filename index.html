SKILLS

P5.JS

An interactive carousel built with Python and MediaPipe hand tracking technology. This project reflects my ongoing exploration of making digital experiences more engaging. As marketing evolves toward inclusivity and participation, I've been investigating how thoughtful design and intuitive interaction can create meaningful connections between users and content.

TOOLS GENERATOR

The Tools tab on my weekly page offers three creative generators: an ASCII filter, a cross stitch pattern creator, and a Weekly ID generator. The Weekly ID feature uses your date of birth and an uploaded image to create a customized ID card inspired by the Weekly Wonder aesthetic. Each generated ID can be exported in an Instagram friendly format for easy sharing.
